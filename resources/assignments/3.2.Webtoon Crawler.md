## 웹툰 목록 읽기

네이버 웹툰 목록을 출력하는 프로그램을 만듭니다.

- 입력: n(10 <= n <= 15)가 입력되고 다음 n개의 줄에 `'mon', 'tue', 'wed', 'thu', 'fri', 'sat', 'sun'`중 하나가 입력됩니다.
- 출력: 한 줄에 해당 요일에 맞는 웹툰 목록을 출력하면 됩니다. 각 웹툰 이름 사이에 ', '가 추가되어 있어야합니다.


- 입력 예시:

`
2
`

`  
mon
`

`
tue
`
